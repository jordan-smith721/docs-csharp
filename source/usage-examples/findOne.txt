===============
Find a Document
===============

.. default-domain:: mongodb

.. contents:: On this page
   :local:
   :backlinks: none
   :depth: 2
   :class: singlecol

You can retrieve a single document from a collection by using the ``Find()``
method.

Find a Document Using Builders
------------------------------

The following example uses builders to find a document in the ``movies``
collection with the ``title``, "The Great Train Robbery".

.. literalinclude:: ../includes/generated/FindOne.snippet.find-one-builders.cs
   :language: csharp
   :copyable: true

.. io-code-block::
   :copyable: true

   .. input:: ../includes/generated/FindOne.snippet.find-one-builders.cs
      :language:  csharp

   .. output::
      :language:  json
      :visible: false

      { "title" : "The Great Train Robbery", "year" : 1903, "cast" : ["A.C. Abadie", "Gilbert M. 'Broncho Billy' Anderson", "George Barnes", "Justus D. Barnes"], "genres" : ["Short", "Western"] }
